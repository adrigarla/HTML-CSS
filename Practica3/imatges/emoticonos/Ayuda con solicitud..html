Puedes crear un grupo de botones de radio en HTML utilizando la etiqueta <input> con el atributo type="radio". Aquí te muestro un ejemplo de cómo crear un grupo de radio con cuatro opciones:

html
<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Opciones de Radio</title>
</head>
<body>

<form>
  <input type="radio" id="opcion1" name="opciones" value="opcion1">
  <label for="opcion1">Opción 1</label><br>
  <input type="radio" id="opcion2" name="opciones" value="opcion2">
  <label for="opcion2">Opción 2</label><br>
  <input type="radio" id="opcion3" name="opciones" value="opcion3">
  <label for="opcion3">Opción 3</label><br>
  <input type="radio" id="opcion4" name="opciones" value="opcion4">
  <label for="opcion4">Opción 4</label><br>
</form>

</body>
</html>

En este código, cada <input> tiene un type="radio" para indicar que es un botón de radio. La opción name debe ser la misma para todos los botones dentro de un grupo para que el navegador sepa que forman parte del mismo conjunto de opciones. El atributo value especifica el valor que se enviará al enviar el formulario, y la etiqueta <label> proporciona el texto que acompaña a cada botón de radio. La for de cada etiqueta <label> debe coincidir con el id del <input> correspondiente para asociar correctamente el texto con el botón de radio.
